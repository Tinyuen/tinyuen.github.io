4. 一劳永逸的方法

其实不难发现，虽然SASS能够减轻我们很多的开发负担，但是每次写p(20)这样的代码还是会浪费很多时间。有没有一劳永逸的方法呢？
答案是有的。如果你开发中使用Webpack作为构建工具，那么postcss-plugin-px2rem插件（详见 插件文档)就会是个好的选择。这样你就可以摆脱频繁写css函数的困扰，直接按照设计稿标注的px值来写就好了。
